How weight conversion works

Weight conversion works by changing a value from one unit scale into another. A common way to do that is to convert everything into a base unit first, then convert from that base unit into the target unit.

This converter uses grams as the base reference, then calculates the matching value in kilograms, ounces, or pounds.

Example

If you want to convert 5 kilograms to pounds:

5 kg = 5,000 g = 11.02 lb

So 5 kilograms is about 11.02 pounds.
